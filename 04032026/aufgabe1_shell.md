# Aufgabe 1: Die Linux Shell

## Was ist eine Shell
Die Shell is wie ein Interpreter. Befehle wie 'ls' oder 'cd' werden interpretiert und dem Kernel gesagt, was zu tun ist.

## CLI vs. GUI
- GUI (Graphical User Inteface): geeignet wenn man Bilder bearteiten will. Braucht aber viel Arbeitspeicher
-CLI (Command Line Interface): sehr schnell und flexible. Damit kann man 100 Ordner auf einmal erstellen

## Standard-Shell
In meinem Ubuntu is die Bash die Standard-Shell. Dass lässt sich mit dem Befehl 'echo $SHELL' prüfen.
