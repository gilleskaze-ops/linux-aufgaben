# Aufgabe 3: Berechtigungen – Kontrolle über Daten

## Die drei Grundrechte
- **r (read):** Datei lesen / Verzeichnisinhalt auflisten.
- **w (write):** Datei ändern / Dateien im Verzeichnis erstellen oder löschen.
- **x (execute):** Datei als Programm ausführen

## Gruppen von Rechten
In Linux gibt es immer drei Blöcke:
1. **User (u):** Der Besitzer.
2. **Group (g):** Eine bestimmte Gruppe von Leuten.
3. **Others (o):** Der Rest der Welt.
